IRIS CLASSIFICATION

Análisis del Dataset

El conjunto de datos Iris incluye 150 ejemplos pertenecientes a tres especies de iris distintas: setosa, versicolor y virginica. Cada ejemplo se caracteriza por cuatro atributos numéricos: longitud del sépalo, ancho del sépalo, longitud del pétalo y ancho del pétalo.
El objetivo fundamental del conjunto de datos Iris es clasificar las muestras de iris según estos cuatro atributos. Es importante destacar que este dataset no contiene valores faltantes ni datos incompletos.
Se han realizado operaciones estadísticas para comprender mejor la distribución de estos datos. Estas operaciones proporcionan información esencial sobre la variabilidad y las tendencias presentes en cada especie de iris en función de sus atributos.
Los resultados de este análisis se presentan a continuación:

Resultados obtenidos mediante el estudio del dataset en Excel:

Resultados obtenidos mediante el estudio del dataset en RapidMiner:

Al comparar los datos estadísticos obtenidos en este ejercicio con los del ejercicio anterior, se observa que, para cada atributo, las medidas estadísticas, como la media, la desviación estándar y la varianza, son idénticas en ambos casos. Además, al observar los gráficos, se nota que los atributos muestran cierto sesgo, principalmente hacia la derecha, lo que sugiere la necesidad de normalizar los datos.
Este análisis preliminar proporciona una base para continuar con el proceso de clasificación y modelado de las especies de iris utilizando algoritmos de machine learning.

A modo de ejercicio se realizaron las operaciones estadísticas utilizando Google Colaboratory, obteniendo los mismos resultados que en los dos ejercicios anteriores, además de generar algunos gráficos extras.
