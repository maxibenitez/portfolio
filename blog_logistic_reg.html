Regresión Logística

La regresión logística es una técnica que se utiliza principalmente para problemas de clasificación binaria, donde debemos predecir entre dos clases posibles. Por ejemplo, si un correo electrónico es spam o no spam. La regresión logística se basa en la función logística, también conocida como función sigmoide. Esta función transforma cualquier número en un valor entre 0 y 1. Es como una curva en forma de "S" y se usa para modelar probabilidades.

Similar a la regresión lineal, la regresión logística utiliza una ecuación con coeficientes para combinar valores de entrada y predecir una salida binaria. Los coeficientes se aprenden a partir de los datos de entrenamiento. Los coeficientes en la ecuación de regresión logística se estiman a través de la estimación de máxima verosimilitud. Este proceso busca los valores de coeficientes que minimizan el error entre las probabilidades predichas y las reales.

La regresión logística predice la probabilidad de que una instancia pertenezca a una clase específica. Hacer predicciones con regresión logística es simple. Simplemente ingresas los valores en la ecuación y calculas la probabilidad. Luego, puedes redondear para obtener una clasificación final.

Preparar Datos para Regresión Logística

  • Variable de salida binaria
  • Eliminar ruido (valores atípicos)
  • Asegurarse que los datos sigan una distribución Gaussiana
  • Eliminar entradas altamente correlacionadas
  • Fracaso en la convergencia (debido al no realizar el punto anterior)
