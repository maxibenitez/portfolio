ANÁLISIS DISCRIMINANTE LINEAL

La regresión logística es un algoritmo de clasificación tradicionalmente limitado solo a problemas de clasificación de dos clases. Si tienes más de dos clases, entonces el Análisis Discriminante Lineal (LDA, por sus siglas en inglés) es la técnica de clasificación lineal preferida.

Limitaciones de la Regresión Logística

La regresión logística es efectiva pero tiene limitaciones:

  • Problemas de Dos Clases: La regresión logística está diseñada para problemas de clasificación de dos clases o binarios. Puede ampliarse para clasificación multiclase, pero rara vez se utiliza para este propósito.
  • Inestable con Clases Bien Separadas: La regresión logística puede volverse inestable cuando las clases están bien separadas.
  • Inestable con Pocos Ejemplos: La regresión logística puede volverse inestable cuando hay pocos ejemplos para estimar los parámetros.

Representación de los Modelos LDA

El LDA se basa en propiedades estadísticas de los datos para cada clase, como las medias y las matrices de covarianza. Estos valores se utilizan para hacer predicciones y se almacenan en el modelo aprendido.

Aprendizaje de Modelos LDA

El LDA asume que los datos siguen una distribución gaussiana y que cada atributo tiene la misma varianza. Estima la media y la varianza para cada clase a partir de los datos, utilizando fórmulas para calcular estos valores.
$$mean_k = \frac {1}{n_k} \times \sum_{i=1}^n x_i$$
Donde meank es el valor medio de x para la clase k, nk es el número de instancias con la clase k. La varianza se calcula en todas las clases como la diferencia promedio al cuadrado de cada valor con respecto a la media.
$$sigma^2 = \frac {1}{n - K} \times \sum_{i=1}^n(x_i-mean_k)^2$$
Donde sigma^2 es la varianza en todas las entradas (x), n es el número de instancias, K es el número de clases y meank es la media de x para la clase a la que pertenece xi.

Realizando Predicciones con LDA

El LDA realiza predicciones calculando la probabilidad de que una entrada pertenezca a cada clase y utilizando el Teorema de Bayes. Utiliza funciones discriminantes para estimar las probabilidades y clasificar nuevas instancias en función de las probabilidades estimadas.
$$D_k(x) = x \times \frac{mean_k}{sigma^2} - \frac{(mean_k)^2}{2\times sigma^2} + ln(P(k))$$
Dk(x) es la función discriminante para la clase k dada la entrada x; meank, sigma2 y P(k) se estiman a partir de tus datos. La función ln() es el logaritmo natural.

Preparación de Datos para LDA

  • Considerar que sean problemas de clasificación
  • Asegurarse que los datos sigan una distribución Gaussiana
  • Eliminar ruido (valores atípicos)
  • Estandarizar los datos para que tengan media 0 y desviación estándar 1 (misma varianza)
