MATPLOTLIB

MATPLOTLIB

En esta entrada, podrás explorar una de las librerías más utilizadas en la visualización de datos. Podrás visualizar una lista de sus funciones más importantes, junto con los parámetros que puedes ajustar para perfeccionar los gráficos y visualizaciones.

Creación de gráficos

  • plt.plot(): Crea un gráfico de líneas.
    • x y y: Datos a graficar.
    • label: Etiqueta para la serie de datos.
    • color: Color de la línea.
    • linestyle: Estilo de línea.
    • marker: Marcador para puntos de datos.
  • plt.scatter(): Crea un gráfico de dispersión.
    • x y y: Datos a graficar.
    • color: Color de los puntos.
    • marker: Marcador para puntos de datos.
  • plt.bar(): Crea un gráfico de barras verticales.
    • x y height (o y y width): Datos de barras.
    • color: Color de las barras.
    • width: Ancho de las barras.
  • plt.barh(): Crea un gráfico de barras horizontales.
    • x y height (o y y width): Datos de barras.
    • color: Color de las barras.
    • width: Ancho de las barras (para plt.bar()).
  • plt.hist(): Crea un histograma.
    • x: Datos para el histograma.
    • bins: Número de divisiones del histograma.
  • plt.boxplot(): Crea un gráfico de caja y bigotes.
    • x: Datos para el gráfico de caja y bigotes.
  • plt.pie(): Crea un gráfico circular de tarta.
    • x: Datos para el gráfico circular de tarta.
    • labels: Etiquetas para cada sector.
    • colors: Colores de los sectores.
  • plt.imshow(): Muestra una imagen en el gráfico.
    • X: Matriz de valores de píxeles para la imagen.
    • cmap: Mapa de colores para la representación de la imagen.
  • plt.contour(): Crea un gráfico de contorno.
    • X y Y: Coordenadas para el gráfico de contorno.
    • Z: Valores de contorno.
  • plt.colorbar(): Agrega una barra de colores a un gráfico.
    • mappable: Objeto de mapeo al que se asocia la barra de colores.

Configuración de gráficos

  • plt.figure(): Crea una nueva figura.
    • figsize: Tamaño de la figura en pulgadas.
  • plt.xlabel(), plt.ylabel(): Agrega etiquetas a los ejes x e y.
    • xlabel: Etiqueta del eje x.
    • ylabel: Etiqueta del eje y.
  • plt.title(): Agrega un título al gráfico.
    • label: Título del gráfico.
  • plt.legend(): Agrega una leyenda al gráfico.
    • labels: Etiquetas de leyenda.
  • plt.grid(): Agrega una cuadrícula al gráfico.
    • which: Tipo de cuadrícula ('major' o 'minor').
  • plt.xlim(), plt.ylim(): Establece los límites de los ejes x e y.
    • left, right, bottom, top: Límites de los ejes x e y.
  • plt.xticks(), plt.yticks(): Establece las marcas y etiquetas de los ejes.
    • ticks: Ubicaciones de las marcas en el eje.
    • labels: Etiquetas de las marcas en el eje.
  • plt.subplots(): Crea subgráficos en una figura.
    • nrows: Número de filas de subgráficos.
    • ncols: Número de columnas de subgráficos.
  • plt.tight_layout(): Ajusta automáticamente los espacios entre subgráficos.

Personalización de gráficos

  • plt.text(): Agrega texto personalizado al gráfico.
    • x y y: Posición del texto en el gráfico.
    • s: Texto a mostrar.
  • plt.annotate(): Agrega una anotación con una flecha al gráfico.
    • s: Texto de la anotación.
    • xy y xytext: Posiciones de la flecha y el texto.
  • plt.savefig(): Guarda el gráfico como un archivo de imagen.
    • fname: Nombre del archivo de imagen.
    • format: Formato de la imagen (por ejemplo, 'png', 'jpg').
