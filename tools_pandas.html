PANDAS

En esta entrada, podrás explorar una de las librerías más utilizadas en el análisis de datos. Podrás visualizar una lista de sus funciones más importantes, junto con los parámetros que puedes ajustar para optimizar las tareas de procesamiento y manipulación de datos.

Manipulación de datos

  • pd.read_csv(): Lee un archivo CSV y lo convierte en un DataFrame.
    • filepath_or_buffer: Ruta o URL del archivo CSV.
    • sep: Separador de campos (por defecto es ",").
    • header: Fila que contiene los nombres de las columnas (por defecto es 0).
    • index_col: Columna que se utilizará como índice (por defecto es None).
  • pd.read_excel(): Lee un archivo de Excel y lo convierte en un DataFrame.
    • io: Ruta o URL del archivo Excel.
    • sheet_name: Nombre de la hoja de Excel a leer.
    • header: Fila que contiene los nombres de las columnas (por defecto es 0).
    • index_col: Columna que se utilizará como índice (por defecto es None).
  • df.head(): Muestra las primeras filas del DataFrame.
    • n: Número de filas para mostrar (por defecto es 5).
  • df.tail(): Muestra las últimas filas del DataFrame.
    • n: Número de filas para mostrar (por defecto es 5).
  • df.shape: Devuelve las dimensiones (número de filas y columnas) del DataFrame.
  • df.info(): Proporciona información sobre el DataFrame, incluyendo tipos de datos y valores no nulos.
  • df.describe(): Calcula estadísticas descriptivas para columnas numéricas.
    • percentiles: Lista de percentiles a calcular (por defecto es [0.25, 0.50, 0.75]).
  • df.drop(): Elimina filas o columnas del DataFrame.
    • labels: Nombre o lista de nombres de filas o columnas a eliminar.
    • axis: Indica si se elimina una fila (axis=0) o una columna (axis=1).
  • df.rename(): Cambia los nombres de columnas.
    • mapper: Diccionario o función que especifica cómo se renombrarán las columnas.
  • df.sort_values(): Ordena el DataFrame por columnas específicas.
    • by: Nombre o lista de nombres de columnas por las que se ordenará.
    • ascending: Indica si la clasificación es ascendente (True) o descendente (False).
  • df.groupby(): Agrupa los datos por una o más columnas.
    • by: Nombre o lista de nombres de columnas por las que se agruparán los datos.
  • df.pivot_table(): Crea una tabla dinámica a partir de los datos.
    • values: Nombre o lista de nombres de columnas a agregar.
    • index: Nombre o lista de nombres de columnas para los índices de la tabla dinámica.
    • columns: Nombre o lista de nombres de columnas para las columnas de la tabla dinámica.
  • df.merge(): Combina dos DataFrames por columnas comunes.
    • right: DataFrame con el que se fusionará.
    • how: Tipo de fusión (por defecto es 'inner').
    • on: Nombre o lista de nombres de columnas comunes.
  • df.concat(): Concatena DataFrames vertical u horizontalmente.
    • objs: Lista de DataFrames a concatenar.
    • axis: Indica si la concatenación es vertical (axis=0) u horizontal (axis=1).
  • df.to_csv(): Guarda el DataFrame en un archivo CSV.
    • path_or_buf: Ruta o objeto de archivo para guardar el CSV.
    • sep: Separador de campos (por defecto es ",").
  • df.to_excel(): Guarda el DataFrame en un archivo de Excel.
    • excel_writer: Ruta o objeto de archivo para guardar el archivo de Excel.
  • pd.to_datetime(): Convierte una columna en tipo de datos datetime.
    • errors: Manejo de errores al convertir fechas (por defecto es 'raise').

Selección y filtrado de datos

  • df[column_name]: Selecciona una columna específica.
  • df.loc[]: Permite la selección de filas y columnas por etiquetas.
  • df.iloc[]: Permite la selección de filas y columnas por índices numéricos.
  • df[condition]: Filtra filas basadas en una condición.
  • df.query(): Filtra filas usando una expresión de consulta.

Operaciones de datos

  • df.apply(): Aplica una función a lo largo de un eje del DataFrame.
    • func: Función a aplicar a lo largo del eje.
    • axis: Indica si la función se aplica a lo largo de filas (axis=0) o culumnas (axis=1).
  • df.fillna(): Rellena valores nulos o faltantes con un valor específico.
    • value: Valor para rellenar los valores nulos.
  • df.drop_duplicates(): Elimina filas duplicadas.
  • df.replace(): Reemplaza valores en el DataFrame.
    • to_replace: Valor o lista de valores que se reemplazarán.
    • value: Valor de reemplazo.
  • df.groupby().agg(): Realiza agregaciones personalizadas en grupos de datos.
    • aggfunc: Función de agregación personalizada a aplicar.

Operaciones estadísticas

  • df.mean(): Calcula la media de las columnas numéricas.
  • df.median(): Calcula la mediana de las columnas numéricas.
  • df.std(): Calcula la desviación estándar de las columnas numéricas.
  • df.corr(): Calcula la matriz de correlación entre columnas numéricas.
